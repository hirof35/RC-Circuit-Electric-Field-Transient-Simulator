# RC Circuit & Electric Field Transient Simulator (Pure Go)

動電気（回路の過渡現象）と静電気（電場の時間変化）という、2つの物理現象の動向を1つのタイムライン上で結合し、リアルタイムにアニメーションGIFとして視覚化する純Go言語製の物理シミュレーターです。

外部のC言語コンパイラ（Cgo/GCC）やGUIライブラリへの依存を一切排除し、Goの標準ライブラリ（`image`, `image/gif`, `math`）のピクセル操作と高速な並列・逐次演算のみで堅牢に動作します。

## 🔬 シミュレーションの物理モデル

### 1. 動電気側：RC直列回路の過渡現象（Transient Phenomena）
回路に電圧 $V_{\text{source}}$ を印加した瞬間からの、コンデンサへの電荷蓄積をオイラー法（Euler method）を用いて微分方程式を逐次計算します。
* **回路電流 $I$ の時間減衰:** $I = \frac{V_{\text{source}} - V_c}{R}$ （ただし $V_c = \frac{Q}{C}$）
* **電荷 $Q$ の更新:** $Q_{t+\Delta t} = Q_t + I \times \Delta t$

### 2. 静電気側：電荷に同期する電場（Electric Field）の変化
回路の各ステップで蓄積されたリアルタイムの電荷量 $Q$ をそのまま空間の光源（点電荷）のエネルギーへとフィードバックします。
* **電場強度 $E$:** 距離の2乗に反比例し、電荷 $Q$ に比例する物理挙動をピクセル単位のグラデーション（アルファ値および輝度）としてマッピングします。
